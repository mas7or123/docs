# EC Anti-Cheat Documentation

This guide explains how to install, configure, and manage the EC Anti-Cheat system and its dashboard integration.

---

## Startup

### Download & Install

- Download the anti-cheat from the official store or dashboard.
- You will receive a ZIP file containing the resource folder.

### Installation Steps

1. Extract the ZIP file.
2. Drag and drop the folder into your server `resources` directory.
3. **Do not rename** the folder or any files inside it.
4. Open `server.cfg`.
5. Ensure the anti-cheat **as the first resource**:

```cfg
ensure EC_AC
