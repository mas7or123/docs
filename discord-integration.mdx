# Discord Integration

<img src="/images/discord.svg" alt="Discord" />

## Link your Discord server

<Steps>
  <Step title="Open dashboard">
    Go to **Settings → Discord Integration**.
  </Step>
  <Step title="Connect server">
    Click **Connect Server** and authorize.
  </Step>
  <Step title="Confirm">
    Dashboard shows: **Connected** (replace with your real UI labels later).
  </Step>
</Steps>

## Invite the bot

<Button href="https://discord.com/oauth2/authorize?client_id=000000000000000000&permissions=8&scope=bot%20applications.commands" icon="bot">
Invite Discord Bot
</Button>

<Note>
Replace the `client_id` in the invite URL with your real bot application ID.
</Note>

<Warning>
Give the bot **Administrator**. If you restrict it, webhooks and command actions will fail or partially work.
</Warning>

## Create the staff role

1. Discord → Server Settings → Roles
2. Create role: `Eagle`
3. Assign it to admins/mods

<img src="/images/roles.svg" alt="Roles" />

## Create webhooks

Run:

```text
/createwebhooks server: "main"
```

This creates channels + webhooks for:
- Logs
- Detections
- Screenshots
- Alerts

<Check>
After setup, you should see logs appear when the server starts.
</Check>
