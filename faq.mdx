# FAQ

<img src="/images/faq.svg" alt="FAQ" />

## How to turn off global bans

<Tip>
Use global bans only if you trust your network rules and appeal flow. Otherwise keep it off.
</Tip>

**Dashboard → Settings → Security → Global bans → Disable**

## How to unban or whitelist a global ban

<Tabs>
  <Tab title="Whitelist (recommended)">
- Dashboard → Global bans → Search player → **Whitelist**
- Add a note for staff audit
  </Tab>
  <Tab title="Support review">
- Open a support ticket
- Provide identifiers + evidence
  </Tab>
</Tabs>

## How to unban a spoofer ban

<Warning>
If you unban spoofers blindly, they come back instantly. Do manual review.
</Warning>

Typical flow:
1. Verify evidence
2. Confirm identifiers
3. Unban in dashboard
4. Optional: add whitelist entry

## How to change the server IP

1. Change IP on host
2. Update dashboard server settings
3. Restart FXServer

## Does Eagle work on Linux?

Yes.

<Check>
If FXServer runs, Eagle runs. Linux issues are usually file permissions, not compatibility.
</Check>
