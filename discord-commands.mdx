<Info>
Argument names below are examples. Replace them with your real slash-command schema.
</Info>

## Command list

| Command | What it does | Typical permission |
|---|---|---|
| `/ban` | Ban a player | Admin |
| `/unban` | Remove a ban | Admin |
| `/screenshot` | Capture player screen | Moderator |
| `/createwebhooks` | Setup channels/webhooks | Owner |

---

## /ban

### Arguments

| Name | Type | Required | Example |
|---|---|---:|---|
| `id` | number | Yes | `23` |
| `reason` | string | Yes | `Cheating` |
| `duration` | string | No | `7d`, `12h`, `perm` |
| `silent` | boolean | No | `true` |
| `proof` | string | No | URL to clip/screenshot |

### Examples

<CodeGroup>
```text
/ban id: 23 reason: "Cheating"
```
```text
/ban id: 23 reason: "Aimbot" duration: "7d" proof: "https://clips.example.com/abc"
```
```text
/ban id: 23 reason: "Injected menu" duration: "perm" silent: true
```
</CodeGroup>

## /unban

### Arguments

| Name | Type | Required | Example |
|---|---|---:|---|
| `identifier` | string | Yes | `license:xxxxxxxx` |
| `reason` | string | No | `Appeal accepted` |
| `whitelist` | boolean | No | `true` |

### Examples

```text
/unban identifier: "license:xxxxxxxx" reason: "Appeal accepted"
```

## /screenshot

### Arguments

| Name | Type | Required | Example |
|---|---|---:|---|
| `id` | number | Yes | `23` |
| `note` | string | No | `Check UI overlays` |

### Example

```text
/screenshot id: 23 note: "Suspected ESP"
```

<Note>
Route screenshots to a dedicated channel so evidence doesn’t get buried.
</Note>

## /createwebhooks

### Arguments

| Name | Type | Required | Example |
|---|---|---:|---|
| `server` | string | No | `main` |
| `category` | string | No | `Eagle Logs` |
| `private` | boolean | No | `true` |

### Example

```text
/createwebhooks server: "main" category: "Eagle Logs" private: true
```

<Warning>
Only run this with proper bot permissions. Otherwise you’ll get partial setup and missing logs.
</Warning>
