## Framework selection

You must set **both** correctly.

### Files to edit

- `resources/[EC]/EC_AC/configs/shared_config.lua`
- `resources/[EC]/EC-PANEL/config.lua`

---

<Steps>
  <Step title="Configure EC_AC">
    Open:

    ```text
    resources/[EC]/EC_AC/configs/shared_config.lua
    ```

    Set the framework:

    <Tabs>
      <Tab title="ESX">
      ```lua
      EG_AC.Core = "es_extended"
      ```
      </Tab>

      <Tab title="QBCore">
      ```lua
      EG_AC.Core = "qb-core"
      ```
      </Tab>

      <Tab title="Standalone">
      ```lua
      EG_AC.Core = "other"
      ```
      </Tab>

      <Tab title="vRP">
      ```lua
      EG_AC.Core = "vrp"
      ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Configure EC-PANEL">
    Open:

    ```text
    resources/[EC]/EC-PANEL/config.lua
    ```

    Set the core:

    <Tabs>
      <Tab title="ESX">
      ```lua
      Config.Framework = "es_extended"
      ```
      </Tab>

      <Tab title="QBCore">
      ```lua
      Config.Framework = "qbcore"
      ```
      </Tab>

      <Tab title="Standalone">
      ```lua
      Config.Framework = "standalone"
      ```
      </Tab>

      <Tab title="vRP">
      ```lua
      Config.Framework = "vrp"
      ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
